// Agent System
Dual-Agent Architecture
research_agent.py
LangGraph Agent / ReAct Loop
主 Agent，基于 LangChain/LangGraph 构建。支持多工具调用：爬取网页、搜索 arXiv、抓取 Twitter 书签、查询 wiki、保存分析报告。MiniMax M2.7 驱动，OpenAI 兼容接口。
main.py / ResearchMission
Strategy Router / CLI
批量处理入口，自动路由策略。单条记录 → DeepStrategy（深度分析）；多条记录 → ClusterStrategy（聚类处理）。Twitter 书签一键抓取并分析。
